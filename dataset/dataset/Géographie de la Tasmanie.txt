 Île Badger ;
 Île Betsey ;
 Île Bruny ;
 Île Christmas ;
 Flinders Island ;
 Île Hogan ;
 Hope Island ;
 Île Hunter ;
 Île King ;
 Long Island ;
 Île Maria ;
 Pedra Branca ;
 Île aux Perdrix ;
 Île Schouten ;
 Île Taillefer ;
 Île Tasman.
